# README

# カードデザイン型掲示板App

## 機能

- 掲示板の基本機能　(閲覧,新規投稿,編集,削除)
- ユーザ登録, ログイン機能
- 掲示板は2種類あります
    - 共通掲示板: 未ログインユーザ, ログインユーザ共に使用可能
    - ユーザ掲示板: ログインしたユーザしか見れない掲示板。未ログインユーザからはアクセスできない

##  推しポイント
柴犬のフリー画像<br>
画像選択時のアクション(JQueryで実装)
